# House Property Management System (Oracle SQL Project)

## Description
This project is designed to manage house buying and selling properties using Oracle SQL.

## Features
- Property listing
- Seller property tracking
- Highest price analysis
- Second highest price analysis
- Available property search

## Technologies
Oracle SQL

## SQL Concepts Used
- Joins
- Aggregate Functions
- Subqueries
- Group By
- Having

## Future Scope
- HTML frontend
- CSS styling
- JavaScript interaction
- Python database connection
